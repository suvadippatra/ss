What is this / এটি কী?
📝

প্রশ্নপত্র Maker creates exam papers with a two-set A4 landscape print layout. Everything runs locally.
এটি একটি দুই-সেট A4 ল্যান্ডস্কেপ প্রশ্নপত্র তৈরি করার টুল। সবকিছু লোকালি কাজ করে।

✍️ Formatting Rules / ফরম্যাটিং নিয়ম (Manual Typing)
→

[EQ:\frac{1}{2}] : Math equations
গণিতের সমীকরণ রেন্ডার করে।

→

[CHEM:H2O] : Chemistry formulas
রাসায়নিক সংকেত (অটো সাবস্ক্রিপ্ট)।

→

x[SUP:2] : Superscript (x²)
সুপারস্ক্রিপ্ট লেখার জন্য।

→

C[SUB:3] : Subscript (C₃)
সাবস্ক্রিপ্ট লেখার জন্য।

→

[SYM:alpha] : Greek symbols
গ্রিক অক্ষর (যেমন: আলফা, বিটা)।

Beginner's Guide / নতুনদের গাইড
1. 📝 Setup Details / বিস্তারিত সেটআপ
→

Paper Title / পেপারের নাম: The main heading.
প্রধান শিরোনাম (যেমন: Weekly Test 1)।

→

Class, Time, & Subject: Full Marks automatically calculates.
এগুলো পূরণ করুন। Full Marks স্বয়ংক্রিয়ভাবে হিসাব হবে।

→

Language Mode / পেপার ভাষা: Automatically changes numbering style (১, ২, ৩ vs 1, 2, 3).
ভাষা নির্বাচন করলে নম্বরের স্টাইল নিজে থেকেই বদলে যাবে।

2. 📚 Adding Sections & Questions / বিভাগ ও প্রশ্ন যোগ
→

Add Section: Exam papers are divided into sections.
Add Section-এ ক্লিক করে নতুন বিভাগ তৈরি করুন।

→

Marks / Q: Set marks per question.
প্রতিটি প্রশ্নের নম্বর নির্ধারণ করুন।

→

Answer Any / যে কোনো N টি: E.g., type "3" to answer 3 out of 5.
৫টির মধ্যে ৩টি উত্তর দিতে হলে "3" লিখুন।

→

🔗 Link OR (অথবা): Select two questions to join them with an OR.
দুটি প্রশ্নকে 'অথবা' দিয়ে জুড়তে ব্যবহৃত হয়।

3. ⊞ Advanced Tools / উন্নত প্রশ্ন টুলস
→

⊞ Table / টেবিল: Insert typing tables.
টেবিল তৈরি করতে ক্লিক করুন।

→

✏ Diagram / চিত্র: Draw directly on the paper!
সরাসরি ডায়াগ্রাম বা ছবি আঁকুন।

4. 🤖 The AI Magic Trick / এআই ম্যাজিক (OCR)
→

Use 🤖 OCR Prompt to turn photos into ready papers via AI!
প্রম্পট কপি করে ChatGPT/Gemini তে ছবি দিন, আউটপুট ইম্পোর্ট করুন।

5. 🎨 Paper Settings / পেপার সেটিংস
→

Auto-fit: Magically shrinks/expands to fit 1 page!
পেজ ফিট করতে সাইজ ও মার্জিন অটো ঠিক করে।

→

Watermark / জলছাপ: Add your school logo to the background.
স্কুলের লোগো আপলোড করুন।

6. 🖨️ Exporting & Printing / এক্সপোর্ট ও প্রিন্ট
→

.qpf: Saves a backup file! Always do this.
ব্যাকআপ ফাইল। কাজ শেষে অবশ্যই সেভ করবেন!

→

Print / PDF: Set to A4, Landscape, Margins: None.
প্রিন্ট করার জন্য A4 ও Landscape নির্বাচন করুন।
